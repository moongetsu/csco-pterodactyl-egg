## Features ✨
- Easy installation process
- Automatic server configuration
- Support for custom maps
- Optimized server settings
- Built-in error handling
- Detailed installation logs
- Support for multiple package managers (apt, yum, apk)

## Requirements 🛠️
- Pterodactyl Panel 1.x
- Docker
- Minimum 20GB of disk space
- Linux-based system

## Installation 👀
1. Download the egg file
2. Import it into your Pterodactyl panel
3. Create a new server using this egg
4. Configure the required variables:
   - Default Map (e.g., de_dust2_csco)
   - Steam Game Server Login Token
   - Maximum Players
   - Server Files Download URL (optional)

## Configuration Variables 🧐
| Variable | Description | Default Value |
|----------|-------------|---------------|
| Default Map | Initial map to load | de_dust2_csco |
| Steam Game Server Login Token | Steam authentication token | - |
| Maximum Players | Max players allowed | 32 |
| Server Files Download URL | Custom download URL | https://cdn.moongetsu.ro/csco/csco.zip |

## Server Settings ⚙️
The egg includes optimized server settings for:
- Network rates
- Tickrate
- Interpolation
- Download/Upload settings
- GOTV configuration
- Performance optimization
